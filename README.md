# RxF1 - HeroDevs

Web application with information about Formula 1 Championship since 2018 to 2022. The information about it is provided by [Formula One API by Ergast](http://ergast.com/mrd).

## Development server

Run `nx serve rxf1-hero-devs` for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
